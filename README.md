# What is this project about?
This is a simple project where I played with custom libraries creation, 
importing and usage in plain Java and Spring Boot project.

For Plain Java it is a simple library that is used in a corresponding 
App project;

For Spring Boot it is a starter, that consists of core functionality, 
autoconfiguration, the starter that simplifies usage of library and the 
parent project that simplifies modules building and compiling.
In a corresponging demoApplication I use this starter.
